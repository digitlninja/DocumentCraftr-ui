# Description 
The UI for Keenious ReCraftr Application

Built with ReactJS and React-Redux.

## Running the app

Build the docker container. Run this command from the project root folder.
```
$ docker build -t ui .
```

Spin up the container
```
$ docker run -it -p 3000:3000 ui
```

Enjoy!
